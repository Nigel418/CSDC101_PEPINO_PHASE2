# CSDC101_PEPINO_PHASE2
How to Run

1. To run and play the full fan made game, click on the Pepino_FINAL file.

2. If you want to use the training loop, run the trainingloop file

3. if you want to use the battleloop immediately with predefined stats, run the battleloop file.
